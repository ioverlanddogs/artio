# IMPLEMENTATION_PLAN

## Phase 1: creator foundation
Goal:
Allow artists and venue owners to publish and manage content confidently.

Build order:
1. Creator account model and routing
2. Creator onboarding
3. Dashboard
4. Draft models and CRUD APIs
5. Event creation stepper
6. Gallery creation stepper
7. Media upload abstraction
8. Preview pipeline
9. Publish flow and state handling
10. Public page config basic version
11. Creator analytics summary
12. QA and accessibility pass

## Phase 2: polish and growth
Goal:
Improve creator control, scheduling, and feedback.

Build order:
1. Scheduled publishing
2. Advanced validation and resumable drafts
3. Public page ordering and featured content controls
4. Enhanced analytics ranges and top content
5. Venue-owner-specific configuration improvements
6. Hardening and analytics verification

## Non-goals
- Admin moderation panels
- Complex team collaboration unless repo already supports it
- Full CMS with arbitrary blocks
